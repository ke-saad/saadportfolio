Project Highlights

Let’s take a look at some of my notable projects, showcasing the application of various skills and technologies.

Cashtrack finance manager

CashTrack is built using a modern tech stack for optimal performance and user experience. The frontend utilizes Next.js for server-side rendering, while the backend is powered by Hono for efficient API development. Data is managed with PostgreSQL, and Drizzle ORM simplifies database interactions. Zod ensures secure data validation, and Clerk provides robust user authentication. This combination of technologies enables CashTrack to effectively support personal finance management.

Github repository: https://github.com/ke-saad/CashTrack

SafeZone Web and Mobile App for personal Security

SafeZone is a real-time monitoring application that enhances user safety using geolocation technology to notify users when they enter risky areas. It features an administrative interface for managing secure and dangerous zones, allowing quick visualization of safe (green) and dangerous (red) areas. Built with React.js for the web and React Native for mobile, SafeZone supports administrators in zone management and provides users with real-time alerts. The backend, powered by Node.js and MongoDB, ensures secure data processing and storage.

Github repository: https://github.com/ke-saad/safezone

Shop4ever E-Commerce Application

Shop4ever is a full-stack e-commerce application built on the MERN stack, providing a seamless shopping experience with robust user account management, efficient shopping cart functionality, and advanced product browsing features. Integrated with the Paytm payment gateway, it ensures secure transactions during checkout. The application also includes a powerful admin dashboard for managing orders, products, and user accounts, making it an all-in-one solution for online retail.

Github repository: https://github.com/ke-saad/shop4ever

Flight Manager

The Flight Manager project employs a versatile tech stack ensuring robustness across different platforms. The backend is powered by a Java-based solution with Maven, providing server-side processing and API development. The frontend utilizes modern JavaScript frameworks for building a dynamic web interface with Tailwind CSS for a responsive design.

Github repository: https://github.com/ke-saad/flightmanager

Vogue App

The Vogue App project leverages a solid technology stack to deliver a comprehensive shopping experience for clothing. The backend is built on a Flutter framework with Firebase for cloud storage and authentication. The frontend utilizes Flutter's UI component classes to create a responsive mobile interface, complemented by custom styling that enhances user interaction. This architecture ensures scalability and streamlines the process of uploading, retrieving, and managing clothing items, while incorporating AI functionalities to personalize the shopping experience and optimize user engagement..

Github repository: https://github.com/ke-saad/vogue_app
